# fo Documentation

This directory contains documentation for the fo project, organized by purpose.

## Directory Structure

- **[adr/](adr/)** - Architecture Decision Records (ADRs)
  - Documents key architectural decisions and their rationale
  - See [adr/README.md](adr/README.md) for details

- **[design/](design/)** - Design system documentation
  - Pattern type specifications
  - Visual design guidelines
  - Architecture diagrams

- **[guides/](guides/)** - User and contributor guides
  - GitHub issue templates and prompts
  - Documentation policies
  - How-to guides

- **[archive/](archive/)** - Obsolete or historical documents
  - Analysis documents for closed issues
  - Legacy documentation
  - Historical project snapshots

## Root Files

- **VISION_REVIEW.md** - Product vision and design philosophy
- **RESEARCH_FOUNDATIONS.md** - Research citations and methodology for design decisions
- **TUFTE_PRINCIPLES.md** - Documentation of Tufte design principles applied in fo
- **issues.json** - Issue tracking metadata (for tooling)
